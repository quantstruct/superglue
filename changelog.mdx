# Changelog (2025-02-16 - 2025-02-17)

## Overview
This update introduces significant enhancements to the data transformation and schema generation processes, particularly within the GraphQL API. It also includes improvements in API configuration through AI integration, documentation updates, and refinements in file parsing utilities. These changes aim to enhance system robustness, flexibility, and user experience.

## 🚀 New Features
• Enhanced GraphQL API with a new `instruction` field in the `TransformConfig` type, improving data transformation capabilities. [🔗](https://github.com/quantstruct/superglue/commit/ac416f0c93d8b2b3f525f39252f22cf7ef7ab83a)
• Integration of OpenAI's chat completions for dynamic API configuration, reducing manual errors and improving adaptability. [🔗](https://github.com/quantstruct/superglue/commit/c8e56f43afea8b5b38741d6dbf385856d34cc75a)
